# TechLab Ecommerce - Proyecto Final

Este proyecto es un sistema de e-commerce full stack desarrollado con Java Spring Boot, HTML, CSS, JS y MySQL

Permite gestionar productos, crear pedidos, validar stock en tiempo real y visualizar el historial de compras.

---

## Tecnologías utilizadas

- Java 24
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- CSS
- JavaScript

---

## Funcionalidades principales

###  Productos
- Agregar productos al carrito
- Ver detalles de productos

###  Carrito 
- Agregar / eliminar productos
- Aumentar y disminuir cantidades
- Validación de stock 
- Precio final del pedido

###  Pedidos
- Creación de pedidos
- Validación de stock en backend
- Estado del pedido: PENDIENTE
- Historial de pedidos
- Descontar stock
